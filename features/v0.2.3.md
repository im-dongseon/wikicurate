# v0.2.3 Feature History

## spreadsheet_ingest_policy

**목적:** XLSX 전체 행 추출 방식을 경량화하고, `.gsheet`(Google Drive 스텁) ingest 정책 신규 도입.

**핵심 설계 결정:**
- XLSX: 전체 행 마크다운 테이블 → 시트명 + 컬럼 헤더 + 행 수(추정치)만 추출. `read_only=True, data_only=True`로 메모리 절약. 최대 5행 탐색으로 빈 1행 대응.
- GSHEET: 3단계 fallback — 서비스 계정 → API 키(공개 시트 전용) → URL 기록(인증 안내 삽입)
- `get_values("A1:Z5")` 단일 API 호출로 헤더 탐색 — Google API 할당량 최소화
- XLSX는 `raw/` 중간 파일 없이 wiki 직접 작성 (이전엔 `raw/<이름>.md` 생성 방식이었음)

**변경 파일:** `_system/wiki-schema.md`(XLSX 수정, GSHEET 추가), `_system/commands/setup.md`(gspread 의존성 추가)
